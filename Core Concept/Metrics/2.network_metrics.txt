Network Metrics ใน k6

    timings.dns
        เวลาในการค้นหา DNS (Domain Name System)
        ใช้เมื่อมีการเชื่อมต่อกับโฮสต์ใหม่

    timings.connecting
        เวลาในการเชื่อมต่อ TCP กับเซิร์ฟเวอร์

    timings.tls_handshaking
        เวลาในการดำเนินการ TLS Handshake (สำหรับการเชื่อมต่อ HTTPS)

    timings.waiting
        เวลารอการตอบกลับจากเซิร์ฟเวอร์ (Time to First Byte - TTFB)

    timings.receiving
        เวลาในการถ่ายโอนข้อมูลจากเซิร์ฟเวอร์ถึงไคลเอนต์

การวิเคราะห์ผลลัพธ์

    DNS Lookup สูงผิดปกติ → อาจมีปัญหากับเซิร์ฟเวอร์ DNS
    TCP Connection และ TLS Handshake สูง → อาจมีปัญหากับความเสถียรของเครือข่าย
    TTFB สูง → เซิร์ฟเวอร์อาจใช้เวลามากในการประมวลผลคำขอ
    Content Transfer Time สูง → อาจเกิดจากการส่งข้อมูลขนาดใหญ่

เหมาะสำหรับ:

    การวิเคราะห์ประสิทธิภาพเครือข่าย
    การตรวจหาคอขวดในกระบวนการตอบสนองของเซิร์ฟเวอร์
    การปรับปรุงประสิทธิภาพระบบในระดับเครือข่าย
